#Regression tests for existing functionality, to ensure nothing was broken.
!! test
<collectionPage>

this
is
a
test
</collectionPage>
!! result

<p>this<br />

a<br />

</p>
</div>

!! end

!!test
  <collectionPage> with recursive parsing
!!input
<collectionPage>
this ''is'' a '''test'''
</collectionPage>
!! result
<div class="collectionPage">
<p>this <i>is</i> a <b>test</b>
</p>
</div>

!!end


!!test
  <collectionPage> with leading whitespace
!!input
<collectionPage>

   test

</collectionPage>
!! result
<div class="collectionPage">
<p><br />
&#160;&#160;&#160;test<br />
</p>
</div>

!!end
